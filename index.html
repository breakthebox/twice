TWICE

Toolkit for Web-based Interactive Environments

Download this project as a .zip file Download this project as a tar.gz file

Welcome to the official project web page of the Toolkit for Web-based Interactive Environments (TWICE). TWICE provides structures, concepts, as well as functionalities for the development of collaborative applications based on standard web technologies and bases on the Google Web Toolkit (GWT).

Within the next weeks you will find the sources of the toolkit as well as the documentation. The actual release date will be published at this place as soon as it is defined.
